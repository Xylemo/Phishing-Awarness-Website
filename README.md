## Getting Started

### 1. Download VS Code

If you don't have it yet, download and install Visual Studio Code:

👉 [**Download VS Code**](https://code.visualstudio.com/download)

<!-- ![VS Code Download Page](images/vscode-download.png) -->

### 2. Download the Repository

Go to our GitHub repository page and click the green **"Code"** button, then select **"Download ZIP"**.

<!-- ![Download ZIP](images/download-zip.png) -->

Once downloaded:

1. Find the `.zip` file in your **Downloads** folder
2. **Right-click** → **Extract All**
3. Open VS Code
4. Go to **File → Open Folder** and select the extracted folder

### 3. Install the Live Server Extension

To preview the site in your browser as you work:

1. Open VS Code
2. Click the **Extensions** icon in the left sidebar
3. Search for **"Live Server"**
4. Click **Install**

## Viewing the Site Locally

Once Live Server is installed:

1. Open `index.html` in the editor
2. Right-click anywhere in the file
3. Select **"Open with Live Server"**
4. Your browser will open with a live preview of the site at `http://127.0.0.1:5500`

## Submitting Your Work

If you would like to make a commit, submit them as a **Pull Request** using VS Code's built-in Source Control.

### Step by Step

1. Click the **Source Control** icon in the left sidebar (or press `Ctrl+Shift+G`)
2. You'll see a list of files you've changed
3. Hover over each file and click the **`+`** icon to **stage** your changes
4. Type a short, descriptive **commit message** at the top
5. Click the **checkmark ✓** to commit
6. Click **"Publish Branch"** or **"Push"** to send your changes to GitHub
7. Click **"Sync Changes"** to push to the main repo
8. Go to the GitHub repository in your browser you'll see a prompt to **"Create Pull Request"**
9. Add a title and description, then submit it for review

## Staying Up to Date

Before you start working, always pull the latest changes so you're not editing an outdated version.

### Using the Terminal in VS Code

1. Open the terminal: **Terminal → New Terminal** (or press `` Ctrl+` ``)
2. Run:

```bash
git pull
```

This downloads the latest changes from GitHub and merges them into your local copy.
